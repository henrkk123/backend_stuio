Erste Schritte

Kurzer Überblick, wie du Henriktron lokal startest und dein Q&A-Widget einbindest.

Setup

1. Projekt holen

Repo herunterladen oder clonen.

2. Backend starten

`cd backend`, `npm install`, `npm run dev`. LLM-Key in `.env` eintragen.

3. Admin öffnen

Öffne das Studio unter henriktronstudio.pages.dev.

4. Bot konfigurieren

Identität, Branding, FAQ, Verhalten im Studio pflegen, speichern.

5. Snippet kopieren

Im Tab Export das Produktions-Snippet kopieren.

6. Website einbinden

Snippet vor `` in deine Seite einfügen. Fertig.

Was du brauchst

Eigener Server / Hoster

Du hostest Backend + Widget selbst.

LLM-Account

Eigener API-Key (z.B. Groq, OpenAI, Anthropic) in `.env` eintragen.

Optional

Assistent im Studio nutzen, um Texte/Prompts vorzubereiten.
